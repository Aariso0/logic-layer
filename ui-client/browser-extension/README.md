# README — ui-client/browser-extension/
#
# Manifest V3 browser extension for Chrome / Firefox.
# Wraps supported AI agent web UIs and routes their responses through the
# Logic Layer middleware for claim-by-claim verification.
#
# Build:
#   npm install
#   npm run build        # produces dist/ — load this folder as an unpacked extension
#   npm run dev          # watch mode for local development
